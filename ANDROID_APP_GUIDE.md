# TaskFlow Android/Mobile App - Installation Guide

## 📱 Progressive Web App (PWA)

TaskFlow is now available as a **Progressive Web App** that can be installed on Android devices (and iOS) just like a native app from the Play Store!

## ✨ Features

- **📲 Install on Home Screen** - Works like a native app
- **⚡ Fast Loading** - Optimized performance with caching
- **📴 Offline Support** - Work even without internet connection
- **🔔 Push Notifications** - Stay updated with task changes
- **🎨 Native Experience** - Full-screen app without browser UI
- **💾 Auto-Sync** - Synchronizes data when back online

## 🤖 Android Installation

### Method 1: Automatic Install Prompt
1. Open **Chrome** or **Edge** browser on your Android device
2. Navigate to your TaskFlow website (e.g., `https://yourdomain.com`)
3. After 10 seconds, an install prompt will appear at the bottom
4. Tap **"Install Now"** button
5. The app will be added to your home screen

### Method 2: Manual Installation (Chrome)
1. Open TaskFlow in **Chrome** browser
2. Tap the **three-dot menu** (⋮) in the top-right corner
3. Select **"Install app"** or **"Add to Home Screen"**
4. Confirm by tapping **"Install"**
5. Find the TaskFlow icon on your home screen

### Method 3: Manual Installation (Edge)
1. Open TaskFlow in **Edge** browser
2. Tap the **three-dot menu** (⋮) at the bottom
3. Select **"Add to phone"**
4. Tap **"Add"** to confirm
5. The app icon will appear on your home screen

## 🍎 iOS Installation (iPhone/iPad)

1. Open TaskFlow in **Safari** browser (must use Safari)
2. Tap the **Share button** (square with arrow up) at the bottom
3. Scroll down and tap **"Add to Home Screen"**
4. Edit the name if desired, then tap **"Add"**
5. The app icon will appear on your home screen

## 🔧 Troubleshooting

### Install Prompt Doesn't Appear
- Make sure you're using **HTTPS** (not HTTP)
- Clear browser cache and reload
- Wait 10 seconds after page loads
- Try the manual installation method

### App Not Opening After Install
- Check if service worker is blocked
- Verify internet connection
- Reinstall the app
- Clear browser data

### Offline Mode Not Working
- Wait for initial sync after installation
- Visit key pages while online first
- Check browser storage permissions

### Updates Not Appearing
- Close and reopen the app completely
- Clear app cache: Settings → Storage → Clear cache
- Reinstall if necessary

## 🔄 Updating the App

Updates are automatic! When a new version is available:
1. The app downloads it in the background
2. You'll see a brief notification
3. Close and reopen the app to use the new version

Or manually update:
1. Open the app
2. Pull down to refresh on any page
3. New version will load automatically

## 🗑️ Uninstalling the App

### Android
1. Long-press the TaskFlow icon
2. Select "Uninstall" or drag to trash
3. Confirm removal

### iOS
1. Long-press the TaskFlow icon
2. Select "Remove App"
3. Choose "Delete App"
4. Confirm deletion

## 📊 Benefits vs Native App

| Feature | PWA (Our App) | Native App |
|---------|---------------|------------|
| **Installation** | Direct from website | App Store required |
| **Updates** | Automatic, instant | Manual approval, delays |
| **Storage** | ~10MB | 50-100MB+ |
| **Cross-Platform** | Works everywhere | Separate Android/iOS |
| **Development** | One codebase | Multiple codebases |
| **Cost** | No store fees | Apple/Google fees |

## 💡 Tips for Best Experience

1. **Initial Setup**: Browse all sections while online first
2. **Regular Sync**: Open app daily to keep data fresh
3. **Storage**: Keep at least 50MB free on device
4. **Updates**: Don't force-close the app frequently
5. **Battery**: PWAs use less battery than native apps
